# Sprite textures

Полный список файлов, размеры и арт-гайд — см.
[`public/assets/ART_BRIEF.md`](../ART_BRIEF.md).

Здесь лежат сгенерированные изображения для всех сущностей (башни, мебель, разные виды коллег-врагов).
Игра загружает их по ключам из `TEXTURE_ASSETS` (`src/config.ts`).
Если нужно заменить картинку — просто пересохраните её поверх старой с прозрачным фоном.
