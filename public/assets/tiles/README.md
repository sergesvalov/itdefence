# Tile textures

Полный список файлов, размеры и арт-гайд — см.
[`public/assets/ART_BRIEF.md`](../ART_BRIEF.md).

Здесь лежат бесшовные текстуры пола и стен. 
`MainScene.preload()` подхватывает их автоматически по ключам из `TEXTURE_ASSETS` (`src/config.ts`). 
Для замены просто перезапишите файл (40×40, бесшовный).
